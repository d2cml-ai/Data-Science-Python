Python for Data Science

This course offers comprehensive training in Python, covering everything from the fundamentals of the language and version control with Git and GitHub to advanced techniques in data analysis and artificial intelligence. Participants will explore data collection and manipulation through web scraping with Selenium and API consumption, complemented by data visualization with Seaborn and geospatial analysis using dynamic maps. Additionally, the course introduces the use of Large Language Models (LLMs) in AI, highlighting their applications and ethical considerations. With a practical approach, this course prepares participants to develop projects in data science, advanced programming, and intelligent data analysis.

Date Title Description
Week 1 Introduction to Data Science Overview of the data science discipline, its fundamental phases, and its relationship with fields such as statistics and artificial intelligence. Use cases and the typical workflow in data science projects will be reviewed.
Week 2 Version Control with Git and GitHub Fundamentals of version control and its importance in collaborative projects. Creation and management of repositories, branches, merges, and conflict resolution, as well as integration with GitHub for communication and project deployment.
Week 3 Introduction to Python Overview of the basic syntax of the language, installation, and workspace setup. Data types (numerical, strings, lists, dictionaries) and control structures (conditionals and loops).
Week 4 Functions and Classes in Python Code organization through function definition and the application of object-oriented programming principles. Creation of classes, constructors, attributes, and methods.
Week 5 Data Processing in Python Using libraries such as pandas and NumPy for data manipulation, cleaning, and transformation. Exploration of data formats and preparation of training datasets.
Week 6 Graph Visualization: matplotlib and seaborn Generating effective graphical representations for analysis and communication of results. Configuring charts, styles, and design patterns with matplotlib and seaborn.
Week 7 Web Scraping 1: Selenium Extraction of information from dynamic websites using Selenium. Creation of scripts for browser automation and the development of Robotic Process Automation (RPA). Implementation of element location techniques and handling of wait times to optimize interaction with web pages.
Week 8 Web Scraping 2: APIs Introduction to Application Programming Interfaces (APIs) and their role in data retrieval. Consuming APIs in Python and handling response formats (JSON, XML).
Week 9 Geospatial Analysis 1: Static Maps Principles of georeferencing and the use of libraries for visualizing static maps. Creation and customization of maps, as well as the manipulation of spatial data (shapefiles, GeoJSON).
Week 10 Geospatial Analysis 2: Dynamic Maps Expanding geospatial visualization capabilities through tools and libraries that enable the creation of interactive maps. Integration of layers, controls, and animations to dynamically present geographic data.
Week 11 Raster Data Fundamentals of handling raster data (satellite images and elevation models). Reading, processing, and visualizing these files, as well as integrating them with vector data.
Week 12 Introduction to Working with LLMs: Prompt Engineering Basic concepts on configuring and customizing language models. Design of prompts and strategies to guide text generation toward achieving specific results.
Week 13 Introduction to Working with LLMs: Retrieval-Augmented Generation (RAG) Using data retrieval techniques in combination with automated text generation. Implementing methods that allow the model to query information repositories to enhance the generated output.
